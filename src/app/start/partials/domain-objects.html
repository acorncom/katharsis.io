Domain objects

Now we can add a domain object, a sample resource - let’s be it a Project class which will consist of an identifier and a name.

As stated above, accompanying repository need to be added. This sample doesn’t have any DB connectivity for clearance, but it can be added without any problem.

To allow Katharsis operate on the resources, a special class called repository has to be created. To provide a repository for a created Project model, create a class ProjectRepository. Katharsis can scan this class, and using annotations, find out which methods what can do.

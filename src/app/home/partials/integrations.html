Standard frameworks integration

Katharsis can be easily integrated into server side frameworks.

Database integration

By implementing defined repositories and resources it is easy integrate Katharsis with:

  • SQL solutions like Hibernate or JOOQ
  • No-SQL products like MongoDB and other key-value storage
  • In-memory custom storage
  • And many more...

JAX-RS integration

Katharsis uses internally Jackson for JSON manipulation and can be also used in resource's definition.

JAX-RS integration can be applied in less than 5 minutes and it won't break current implementation.

Servlet integration

Katharsis can be used with other frameworks which use Java Servlet API.

Spring integration

Katharsis provide an easy integration with Spring Boot by providing a configuration class.

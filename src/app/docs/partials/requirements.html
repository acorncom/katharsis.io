Requirements

Java version

Java 8 is required to run Katharsis framework.

Dependencies

We try to keep number of Katharsis dependencies to minimum.

Most notable would be Jackson, which we heavily use for providing all the JSON API goodies.

  • katharsis-core:
Dependency Minimum version*
Jackson 2.5.1
Reflections 0.9.9
Typetools 0.4.0
commons-beanutils 1.9.2
  • katharsis-rs:
Dependency Minimum version*
Jersey 2.17

*Minimum version: Dependencies versions, which Katharsis is build on. We do not guarantee compatibility with previous versions.

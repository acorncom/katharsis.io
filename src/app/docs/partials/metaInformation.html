Meta Information

There is a special interface which can be added to resource repositories to provide meta information: io.katharsis.repository.MetaRepository. It contains a single method MetaInformation getMetaInformation(Iterable<T> resources) which return meta information object that implements a marker interface io.katharsis.response.MetaInformation.

If you want to add meta information along with the responses, all repositories, that is those that implement ResourceRepository and RelationshipRepository, must implement MetaRepository.

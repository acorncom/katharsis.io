Servlet integration

There are two ways of integrating katharsis using Servlets:

  1. Adding an instance of AbstractKatharsisServlet
  2. Adding an instance of AbstractKatharsisFilter

Repository supported parameters

Servlet integration allows a developer to pass the following types of parameters in repository methods:

  1. An instance of ServletContext
  2. An instance of HttpServletRequest
  3. An instance of HttpServletResponse

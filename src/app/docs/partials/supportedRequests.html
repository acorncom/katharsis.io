Supported Requests

Currently Katharsis covers most of the JSON:API specification request types. The following table describes available requests that are can accepted:

HTTP method Kind of the request Sample URL Multiplicity of resources
GET resources http://jsonapi.org/format/#fetching-resources multiple
resource http://jsonapi.org/format/#fetching-resources single
relationship http://jsonapi.org/format/#fetching-relationships single
field http://jsonapi.org/format/#fetching-resources single
POST resource http://jsonapi.org/format/#crud-creating single
PUT resource http://jsonapi.org/format/#crud-creating single
PATCH resource http://jsonapi.org/format/#crud-updating single
DELETE resource http://jsonapi.org/format/#crud-deleting single
